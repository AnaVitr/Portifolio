Sobre Mim

Olá! Eu sou a Ana Vitória, uma programadora apaixonada por tecnologia e desenvolvimento web. Atualmente, estou focada em estudar e aprender novas tecnologias para aprimorar minhas habilidades e criar projetos cada vez mais interessantes.

Minha jornada na programação começou com a curiosidade de entender como as coisas funcionam por trás das telas, e desde então venho me dedicando a explorar o vasto mundo do desenvolvimento front-end.

Estou sempre em busca de novos desafios e oportunidades para crescer profissionalmente, aplicando meus conhecimentos em HTML, CSS e JavaScript para criar interfaces modernas, responsivas e acessíveis.

Tecnologias que Utilizo

HTML

Linguagem de marcação utilizada para estruturar e exibir conteúdo na web. É a base de qualquer site, definindo a estrutura e organização do conteúdo.

CSS

Linguagem de estilo usada para definir a aparência e o layout dos elementos HTML em uma página web, criando designs atraentes e responsivos.

JavaScript

Linguagem de programação que permite adicionar interatividade e dinamismo às páginas web, criando experiências ricas e funcionais para os usuários.

Meus Projetos

Monalisa Interativa

Uma recriação interativa da famosa obra de arte usando p5.js. O projeto demonstra habilidades em manipulação de canvas e interação com o usuário, com olhos que seguem o cursor do mouse.

Corrida - Parte 1

Jogo de corrida simples entre dois jogadores controlados por teclado. Demonstra conceitos básicos de game development com JavaScript, incluindo detecção de colisão e controle de estado do jogo.

Corrida - Parte 2

Versão expandida do jogo de corrida com múltiplos jogadores e melhorias visuais. Mostra evolução no desenvolvimento de jogos, com arrays para gerenciar múltiplos elementos e lógica mais complexa.
